Aide du Formulaire FNRASEC Accusé de réception

Version 2.1.44

Ce formulaire permet de créer un accusé de réception dans un nouveau message Winlink.

En cours d'écriture...

Instructions générales :

  • Assurez-vous que les informations sont exactes et à jour
  • Vérifiez vos données avant de soumettre le formulaire
  • La liste des indicatifs par zone est disponible en bas de cette page.
  • Lorsque vous cliquez sur le bouton "Envoyer", les données sont envoyées vers l'application Winlink.

Champs spécifiques :

  • N° AR
    Le numéro que vous souhaitez donner à votre accusé de réception.
  • N° Message
    Le numéro du message est composé de 2 zones :
    • La première zone est le lieu à partir duquel se fait l'opération : RASEC, COGIC, COZ-Zone, Outremer ou COD.
    • La seconde zone est une séquence :

      Cette séquence s'incrémente automatiquement à chaque ouverture d'un formulaire par Winlink.

      Pour le COGIC et les COZ, la séquence est un nombre sur 3 chiffres : 001, 002, ... 100, 101, etc..

      Pour les COD et les RASEC, la séquence un nombre sur 5 ou 6 chiffres (selon le département). Elle est composée du numéro du département de l'ADRASEC suivi de 3 chiffres. Par exemple pour le 13, la séquence sera sur 5 chiffres et pour le 974, la séquence sera sur 6 chiffres : 13001, 13002, ... 974001, 974002, etc..

      Pour configurer le numéro de séquence : Dans Winlink, aller dans le menu Message/Templates Settings :

    Ces éléments sont importants pour la traçabilité des messages.

  • Niveau de confidentialité
    Le niveau de confidentialité du message :
    DR (Diffusion Restreinte) ou Normal
  • Date & Heure
    La date de l'ouverture du formulaire.
    Le bouton permet de changer le format de l'heure en UTC ou locale.
  • Indicatif de la station
    L'indicatif de la station qui envoie le message.
    C'est l'indicatif configuré dans Winlink. Ce champ se remplit automatiquement.
  • Indicatif de l'opérateur
    L'indicatif du radioamateur qui opère la station.
    Sur une opération de longue durée pendant laquelle plusieurs opérateurs se succédent, chacun met son indicatif.
    Ceci dans un objectif de traçabilité.
    NB : Il y a des situations dans lesquelles l'indicatif de la station et l'ndicatif de l'opérateur sont les mêmes
  • Autorité origine
    L'autorité qui a émis le message.
    Ex: COGIC, COD, COZ, etc...
  • Autorité destinataire
    L'autorité qui reçoit le message.
    Ex: COGIC, COD, COZ, etc...
  • Indicatif du destinataire
    L'indicatif du radioamateur qui reçoit le message.
  • Sujet
    Le sujet du message reçu.
    Recopier ici le sujet et le numéro du message reçu, pour lequel vous emettez l'accusé de réception.
  • Commentaire
    Un commentaire sur le message reçu.
    Ex: Message reçu le 30/03/2025, Message reçu partiellement, etc...

Le bouton envoyer permet d'envoyer les données du formulaire dans le message Winlink associé.

 

Pour toute question supplémentaire, veuillez contacter l'administrateur FNRASEC.
